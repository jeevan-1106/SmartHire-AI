# 🤖 SmartHire AI

> AI Resume Analyzer & Job Recommendation System using NLP, Machine Learning and Streamlit.

---

## 📖 Overview

SmartHire AI is an intelligent resume analysis system that helps users analyze resumes, calculate an ATS score, recommend suitable job roles, identify missing skills, and generate a PDF report.

---

## ✨ Features

- 📄 Resume Upload
- 🎯 ATS Resume Score
- 💼 Job Recommendation
- 📊 Skill Gap Analysis
- 💡 Resume Suggestions
- 📑 PDF Report Generation
- 🌐 Streamlit Web Application

- ---

# 🛠️ Tech Stack

- Python
- Streamlit
- Pandas
- Scikit-Learn
- Plotly
- ReportLab

- ---

# 🧠 Machine Learning Concepts Used

- TF-IDF Vectorization
- Cosine Similarity
- Text Preprocessing
- Skill Matching
- Resume Ranking

- ---

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/jeevan-1106/SmartHire-AI.git
```

Move into the project

```bash
cd SmartHire-AI
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
